# WinSweep

**WinSweep** es una aplicación de limpieza inteligente para Windows enfocada en seguridad, recuperación y mantenimiento del sistema.

## Características

- Limpieza de archivos temporales.
- Limpieza de cachés de navegadores.
- Limpieza profunda.
- Papelera de reciclaje.
- Recovery Clean con cápsulas de recuperación.
- Restauración durante 7 días.
- Historial de limpieza.
- Predicción de almacenamiento.
- Mapa de seguridad.
- Duplicados inteligentes.
- Recordatorios semanales.
- Temas visuales.
- Español e inglés.
